# Warehouse

## 1. 데이터 웨어하우스란?
- 회사가 분석을 위해 자사의 정형 데이터를 저장해두는 중앙 저장소
- 조회 와 분석 최적화
- 과거 데이터 저장
- 사용자는 리포트, 대쉬보드, 분석, 머신러닝으로 사용

## 2. 데이터 웨어하우스 사용 이유
- 소스시스템 운영
- OLTP(서비스 DB)와 OLAP(분석) 분리: 서비스 성능 영향 최소화
- 단일 진실 공급원(SSOT): 지표 정의/집계 로직을 한 곳에서 통일 
- 히스토리 관리: 과거 스냅샷/기간 비교/추세 분석
- 성능/확장성: 컬럼형 저장, MPP(분산쿼리), 파티셔닝/클러스터링
- 보안/거버넌스: 권한, 마스킹, 감사로그, 데이터 카탈로그 연계
