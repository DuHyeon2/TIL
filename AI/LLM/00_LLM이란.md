# LLM이란
- Large Language Model : 거대언어 모델로 텍스트 데이터를 학습하여 자연어를 이해하고 생성하는 모델
- 딥러닝 기술을 사용하여 구축되었다
- GPT 같은게 있음


## 1. Tool
- LLM이 필요할 대 호출하는 외부 기능
- ex)
    - 웹 검색 도구 (검색 API 호출해서 결과 가져오기)
    - DB 조회 도구 (SQL 실행 후 결과 리턴)
    - 코드 실행 도구 (Python 코드 실행, 결과 리턴)
- 보통 함수나 api 엔드포인트를 하나의 Tool 로 봄

## 2. Tool Call
- 모델이 나의 질문에 검색이 필요할 때 처럼 상황에 맞는 호출 의사결정을 내려 실제로 도구를 실행하는 단계

## 3. Agent 
- 언제 어떤 도구를 쓸지 스스로 판단해 도구 호출 -> 결과 반영 -> 답변 생성을 반복 실행하는 실행자 

## 4. bind_tools
- LLM에 사용할 도구들을 등록해 도구 호출 가능한 모델로 만드는 LangChain 방식
